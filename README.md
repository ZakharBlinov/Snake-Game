## 📋 ОПИСАНИЕ ПРОЕКТА

**Snake Game** — это классическая аркадная игра «Змейка», разработанная в рамках производственной практики по ПМ.01 «Разработка модулей программного обеспечения для компьютерных систем». Проект представляет собой веб-приложение с двумя режимами игры, системой прогресса, таблицей рекордов и адаптивным дизайном для мобильных устройств.

### Основные возможности:

| Функция | Описание |
|---------|----------|
| 🎮 **Два режима игры** | Режим уровней (6x6, 9x9, 12x12, 15x15, 18x18, 21x21) и классический бесконечный режим |
| 📊 **Система прогресса** | Отслеживание заполнения поля, разблокировка следующих уровней |
| 🏆 **Таблица рекордов** | Сохранение топ-10 результатов в localStorage с именами игроков |
| ⚡ **Динамическая скорость** | Увеличение скорости с каждым съеденным яблоком (настраиваемый параметр) |
| 📱 **Мобильная адаптация** | Экранные кнопки управления, поддержка сенсорного ввода |
| 🌓 **Две цветовые темы** | Переключение между темной и светлой темой оформления |
| 🧪 **Автоматические тесты** | Встроенная система тестирования ключевых функций игры |
| 📝 **Система логирования** | Запись событий игры с возможностью экспорта в JSON |
| 📦 **PWA поддержка** | Установка на мобильные устройства, работа оффлайн |

## 📦 ИНСТРУКЦИЯ ПО ЗАПУСКУ

### Вариант 1: Онлайн (без установки)
1. Перейдите по ссылке: [https://zakharblinov.github.io/Snake-Game/snake.html](https://zakharblinov.github.io/Snake-Game/snake.html)
2. Игра запустится автоматически в браузере

### Вариант 2: Локальный запуск (из репозитория)

#### Предварительные требования
- Любой современный веб-браузер (Chrome, Firefox, Edge, Safari)
- Git (для клонирования репозитория)
- Текстовый редактор (опционально, для изучения кода)

#### Шаги по установке

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/ZakharBlinov/Snake-Game.git
   ```

2. **Перейдите в папку проекта:**
   ```bash
   cd Snake-Game
   ```

3. **Откройте файл в браузере:**
   - **Способ 1:** Дважды кликните на файл `snake.html` в проводнике
   - **Способ 2:** Перетащите файл `snake.html` в окно браузера
   - **Способ 3:** Запустите локальный сервер (рекомендуется для PWA функциональности):
     ```bash
     # Установите live-server глобально (если не установлен)
     npm install -g live-server
     
     # Запустите сервер в папке проекта
     live-server
     ```

### Вариант 3: Установка на мобильное устройство (PWA)

**Для Android:**
1. Откройте [https://zakharblinov.github.io/Snake-Game/](https://zakharblinov.github.io/Snake-Game/) в Chrome
2. Нажмите на меню (три точки) → "Добавить на главный экран"
3. Подтвердите установку

**Для iOS:**
1. Откройте [https://zakharblinov.github.io/Snake-Game/](https://zakharblinov.github.io/Snake-Game/) в Safari
2. Нажмите на кнопку "Поделиться" (квадратик со стрелкой)
3. Выберите "На экран «Домой»"
4. Подтвердите добавление

После установки игра будет работать **оффлайн**!

## 🎮 КАК ИГРАТЬ

### Управление
- **Клавиатура:** стрелки ← ↑ → ↓
- **Мобильные устройства:** экранные кнопки на панели управления

### Правила игры
- Управляйте змейкой с помощью стрелок
- Собирайте красные яблоки для увеличения длины и счета
- Избегайте столкновений со стенами и собственным хвостом
- В режиме уровней необходимо заполнить змейкой всё поле

### Режимы игры

| Режим | Описание | Особенности |
|-------|----------|-------------|
| **📊 Уровни** | 6 уровней сложности (6x6, 9x9, 12x12, 15x15, 18x18, 21x21) | Прогресс-бар, разблокировка уровней |
| **🎯 Классика** | Бесконечная игра на поле 15x15 | Увеличение скорости, рекорды |

### Интерфейс
- **🍎 Счет яблок** — количество собранных яблок / всего яблок на уровне
- **📏 Длина змейки** — текущая длина / максимальная длина
- **🏆 Рекорд** — лучший результат в классическом режиме
- **📊 Прогресс-бар** — заполнение поля (только в режиме уровней)

### Настройки (⚙️)
- **Скорость** — базовая скорость движения (120-450 мс)
- **Ускорение** — на сколько уменьшается интервал за каждое яблоко (1-12 мс)
- **Масштаб** — размер ячеек на поле (45-90 пикселей)

## 🧪 ТЕСТИРОВАНИЕ

В проекте реализована встроенная система автоматического тестирования:

1. Нажмите кнопку **"🧪 тесты"** в интерфейсе игры
2. Запустятся 7 групп тестов:
   - ✅ Создание змейки
   - ✅ Движение змейки
   - ✅ Обнаружение столкновений
   - ✅ Поедание яблока
   - ✅ Смена направления
   - ✅ Границы поля
   - ✅ Сохранение рекордов
3. Отобразится подробный отчет с результатами

**Пример отчета:**
```
🧪 ОТЧЕТ ТЕСТИРОВАНИЯ
📊 15/15 тестов пройдено
Успешность: 100%
```
